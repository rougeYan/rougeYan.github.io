什么是node? npm? 模块?

node

node是JS的服务执行环境,js除了浏览器执行环境之外的另外一个服务环境;前端的很多工具例如webpack打包工具,gulp构建工具等等需要node存在;
说白了就是JS代码除了在浏览器上面运行,还可以在node.js中运行;除此之外,node还具备强大的的功能,Express框架等等

npm

npm是其实就是一个集成代码仓库, 就是在npm仓库里面 去下载各种需要的工具,例如: vue-cli; css预处理器,等等等等;
说白了就是我们在工具箱提取工具的途径;

模块

什么是模块? 就是我们需要用到的代码库;模块可以是我们自己写的js文,一些js函数,也可以是从npm下载的”工具”(例如:vue-router);
当我们需要用到的时候, 头部import就行了;

router中index.js 和 routes 中route.js 其实只是分开两个文件,实际上可以合并;

视频教程中,是合并的写法,但是项目足够大的时候就显得杂乱无章,因此分开写, 在index.js中import routes.js文件,同时routes文件需要export 暴露接口; 其中import export中,ES6有详细的语法说明;

ES6中说道:模块功能主要由两个命令构成：export和import。export命令用于规定模块的对外接口，import命令用于输入其他模块提供的功能。export default命令，为模块指定默认输出。
expoert default 其实相当于整个文件复制粘贴到inde.js下面;
